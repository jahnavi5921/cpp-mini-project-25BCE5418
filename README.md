# 🛒 Inventory Management System (C++)

## 📌 Description
This project is a Shop Stock Controller developed using C++ and Object-Oriented Programming concepts.

## 🚀 Features
- Add new product
- Purchase stock
- Sell stock
- Search product
- Delete product
- Low stock report
- Total inventory value
- File handling (save/load)

## 🧠 Concepts Used
- Classes & Objects
- Encapsulation
- Constructors
- STL (vector)
- File Handling (fstream)

## 🌐 Web Version
This project is compiled to WebAssembly and hosted using GitHub Pages.
